# Proposal: Add Shortcuts tab in Settings

- Change ID: add-settings-shortcuts-tab
- Summary: Add a new tab in the Settings panel that lists the keyboard shortcuts available in the app for quick reference.

## Why
Users asked for an in-app place to discover and recall keyboard shortcuts without leaving the app.

## What Changes
- Extend the Settings panel to include a "Shortcuts" tab with a read-only list of current shortcuts.
